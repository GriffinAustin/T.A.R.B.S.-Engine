# T.A.R.B.S. Engine
A python based game engine. Developed for fun and practice

Pros:
* Easy to use
* Highly and easily customizable
* Cross platform!
* Useful debugging functionality

Cons:
* Only works with python3
* Plenty of bugs
* Probably will always suck
* Inefficient
